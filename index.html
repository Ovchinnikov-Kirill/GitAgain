git + gitHub

Команды гита

git init => Иницировать гит-репозиторий
git status => Сатус репозитория
git log => Просмотр всех коммитов
git add -A => Добавить все файлы перед коммитом
git add . => --||--
git commit -a -m"commit name" => Создать коммитом
git push => Залить на gitHub
git pull => обновить репозиторий (при работе в команде)
git checkout -- index.html => отменить изменения в выбраном файле (до команды add)
git reset -- index.html => отменить изменения в выбраном файле (после команды add)
git reset --hard HEAD^1 => Жесткий откат на один коммит назад(при этом набраный текст удаляеться)
git reset --soft HEAD^1 => ТОже самое, только набраный текст не удаляеться

Работа с ветками branch

git branch => показывает все веткам git branch -v => показывает все веткам последний коммит на каждой ветке git branch develop => создать новую ветку, develop - имя новой созданой ветки, выбираем сами git branch -d develop => удалить выбраную ветку git checkout develop => перейти в новую ветку, develop - имя ветки в которую надо перейти
